# Import Study — Session 2

This folder contains the files required to complete **Step 1: Study Import** during Session 2 of the Genestack × QIAGEN PoC.

In this step, you will:
1. Generate your personal API token in Genestack.
2. Run the provided script to import a study using predefined S3 links.
3. Verify that the imported study appears in your account.

---

## Files in This Folder

### **import-study-script.txt**
This script imports a dataset hosted in S3 storage.  
You only need to:
- replace the placeholder token with your own API token, and  
- run the script in your Python environment.

The imported dataset will serve as the base for all subsequent exercises in this session.

---

## Before Running the Script
- Ensure the Genestack SDK is installed.  
- Confirm your Python environment (Python 3.10).  
- Copy your API token from your user profile in the Genestack interface.
