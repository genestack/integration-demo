# Templates II -  Session 2

This folder contains the resources for **Step 4: Preparing and Importing Templates**.

You will learn how to:
- customise it (add/remove fields),
- import a new version through the SDK,
- apply it to your datasets.

---

## üìÑ Files in This Folder

### **template-script.txt**
A script that:
- reads the template file path from `path-template.json`,
- uploads the customised template,
- applies the template import command using your API token.

Replace the token with your own before running the script.

---

### **SRL-Core-Default-Template.json**
The example customised template used during the session.  
You can:
- modify field names,  
- adjust field types,  
- add constraints or controlled terms.

We will walk through these during the training.

---

### **path-template.json**
Contains:
- the path to the new template,
- identifiers needed by the script,
- optional parameters such as template description.
